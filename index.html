About the bot

Rumble Royale is a text-based Battle Royale Discord bot; every player will be put into a game where they fight against each other untill there is only one winner. The last man standing will be awarded gold for his victory.

With the gold, players can purchase backpacks using the ?shop command. The backpacks can then be opened using the ?open command.
